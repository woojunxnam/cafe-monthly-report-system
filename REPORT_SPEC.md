# Report Specification

목표:

1. 운영 지출을 월간 리뷰 형식으로 정리합니다.
2. 입금 흐름은 매출, 관련자금, 기타 유입으로 분리합니다.
3. 관련자금 그룹 A, 관련자금 그룹 B는 운영비에서 제외하지 않고 별도 관련자금 섹션에서 양방향으로 봅니다.
4. 주요 카테고리는 합계와 개별 거래를 함께 제공합니다.

리포트 기본 섹션:

1. 운영지출 개요
2. 입금 개요
3. 운영지출 카테고리 합계
4. 입금 카테고리 합계
5. 관련자금 그룹 A / 그룹 B 요약
6. 주요 운영지출 개별 거래
7. 주요 입금 개별 거래
8. 관련자금 개별 거래
9. 검토 메모

출력 원칙:

- 표는 흑백 중심
- 그래프는 필요한 범위에서만 색상 허용
- 월별 HTML report 1개와 CSV 보조 파일 여러 개를 생성
