# Report Specification

목표:
1. 운영지출을 월간 리뷰용 형식으로 정리한다.
2. 입금 흐름을 매출성/관련자금/기타유입으로 분리한다.
3. 박재민관련, 성기용관련은 운영비에서 제외하지 않고 별도 관련자금 섹션에서 양방향으로 본다.
4. 주요 카테고리는 합계표와 개별 거래행 표를 함께 제공한다.

리포트 기본 섹션:
1. 운영지출 개요
2. 입금 개요
3. 운영지출 카테고리 합계표
4. 입금 카테고리 합계표
5. 관련자금(박재민/성기용) 요약
6. 주요 운영지출 개별 거래행
7. 주요 입금 개별 거래행
8. 관련자금 개별 거래행
9. 검토 메모

출력 원칙:
- 표는 흑백 중심
- 그래프는 컬러 허용
- 월별 HTML 파일 1개 + CSV 보조 파일 다수
