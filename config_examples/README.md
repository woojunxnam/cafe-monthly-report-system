# Config Examples

이 폴더에는 공개 저장소에 올릴 수 있는 예시 규칙만 둡니다.

실제 운영 규칙은 다음 위치 중 하나를 사용하세요.

- `private/accounts/<account-id>.json`
- 웹 UI에서 계좌별 비공개 규칙 JSON 업로드

예시 파일 목적:

- JSON 구조 설명
- 새 계좌 onboarding
- placeholder 계좌 규칙 초기화

주의:

- 실제 민감 매핑값은 여기에 넣지 않습니다.
- 실제 원본 거래 샘플은 여기에 넣지 않습니다.
- public 배포에서는 이 폴더의 예시만 포함되어야 합니다.
- 실제 운영 규칙은 업로드 방식 또는 로컬 `private/` 방식으로만 사용합니다.
